Edit Ndc

This screen allows users to modify/approve/reject existing or newly created NDC National Drug Code items. 

How to edit an NDC item

1.       Use the Simple or Advanced Search function to find the item.

2.       Click the NDC link associated with the item in the search results table.

3.       Modify the available fields in each section.

4.       Click the submit button.  Note: Some modifications require a second approval.

Additional Actions

Cancel An undo action in a GUI (“cancel” is not an action on an order). button: Returns the user to the search screen without committing any changes.

Save Work in Progress button: Save changes without submitting the change to the database, or initializing a second review action.

Change Product button: Change the parent product for this NDC item.

VA Department of Veterans Affairs Product Name A unique name assigned to each drug product in the National Drug files. link: Opens the “Edit Product” screen for the associated item.

NDC link: Opens the “Details” pop-up screen with additional details associated with the NDC item.
