Edit Home Page

This page allows a user with the supervisor role to change the image and text on the home page of the application.

Home Page Image

This section allows a user to select and/or upload the home page image.  The select image combo-box has three choices, with descriptions of each:

1.       Default System Image – the default image that comes bundled with the application and cannot be changed out for another image without reinstalling the application.  The default image will appear below when this is selected.

2.       Current Custom Image – if a supervisor has previously uploaded a custom image, that image will be used. The custom image will appear below when this is selected.  If a custom image has yet to have been uploaded, the default image will appear.

3.       Upload New Image – allows the supervisor to upload a custom image to the server. This will over-write the previous custom image.  A form field allowing file selection will appear below when this is selected.

The image is displayed on the home page at location #1, see Figure 0‑1.  Images are restricted to a maximum height or width of 500 pixels. Larger images may become scaled and/or cut-off.

Announcement Information

Below is the list of fields that can be changed.  Please note, on the home page, it will always show the text “PBM Pharmacy Benefits Management Office Message A collection of data in a standardized format that can be sent to or received at a given destination.”. The corresponding location is in parenthesis.

·         Title (#2) – this is the text appearing directly above the body

·         Body (#3) – the bulk of the announcement summary

·         Link Title (#4) – the text displayed for the link

·         Link – the actual link URL Uniform Resource Locator.  A value of “#” can be used so that the link won’t go anywhere

A note about uploading a custom image

When selected to upload a custom image, upon clicking the Save Changes button the form will submit.  Once the submission has completed, the select image field will automatically change to Current Custom Image and the image will be displayed below it.
