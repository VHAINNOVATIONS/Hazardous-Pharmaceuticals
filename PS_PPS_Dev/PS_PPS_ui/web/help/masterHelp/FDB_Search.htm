FDB Search

The FDB First DataBank Search is used to search the First Databank Drug Information Framework A set of assumptions, concepts, values, and practices that constitute a way of describing a real situation. for drugs.  Each of the searches will return a list of FDB Packaged Drugs even if the search criteria is for some other drug concept.  For example: the Generic will search the FDB Generic Drug A nonproprietary drug approved by the FDA that is tested against a standard of bioavailability and bioequivalence. concept for matches and then return a list of all the packaged drugs associated with the matched generic drugs.  

The User The requestor, individual, or organization that specifies, or may assist with specification, and typically accepts the project deliverables. There may be multiple users on a given project. Preferences screen holds some FDB Search preferences options. Altering the FDB Search Preferences in the User preference screen may cause any particular search to return different results.

Only the first 501 results from the search will be displayed.

FDB Search Option Type

Search Option Type

Description

NDC National Drug Code

The user will need to enter at least 5 digits and the system will search the first five digits of the 11-digit NDC number of the FDB Packaged Drug.

Label

The text entered will be used to search the FDB Label Name Field.

Generic                                                       

The text entered will be used to search the FDB Generic Drug Name Field.

GcnSeqNo

The GCN Generic Code Number Sequence Number must be a numeric value that corresponds to the GCN Sequence Number of an FDB Generic Drug.

Drug Classifications

The text entered will be used to search the FDB Drug Class The grouping of drug products based on various criteria, which may include similarity of chemical structure, clinical indications, pharmacology, and therapeutic activity. fields for both the FDB Generic and the FDB Dispensable Drug.

All

The All search will search each of the above searches and aggregate the results.
