Perform an Advanced Search

1.       Select the item type to search for.

2.       Apply result filters by selecting the check-box filters.

3.       Select the field to search in from the “where” dropdown list.

4.       Enter or select the value to search by.  Note: Depending on the field selected, values may be entered as text or selected via a drop-down list.

Retrieve Search Templates: Click the button to access a list of saved search templates.  See the Manage Search Templates help topic for additional information.

Item A generic type representing one conceptual entity consisting of a collection of data fields. For example, a medication may contain the following collection of data fields: Product Name, Generic Name, Trade Name, Dosage Form, NDC, etc. Conceptual entities include Medications, Supplies, and Investigationals. Type: Use the dropdown to select the entity type for the search criteria (Product, Orderable Item, NDC National Drug Code)

Item Status: Used to filter “Active” or “Inactive” items

New Item Request A user action that a PEPS Item (OI, Product Item, or NDC Item) addition or modification be considered by the PNM for updates within the PEPS enterprise database.: Used to filter search results by the request status (Approved, Pending, or Rejected)

Where: Select the field to search by.  Each “Item Type” supports a different set of available search fields

Add: Click to add additional search criteria

Exporting Search Results

Click the desired format link in the export panel located below the results template.

Save Search Criteria

Save As: The name of the saved search template

Notes: Optional text to describe the template

Save At: Supervisors may save global templates to be used by “National Managers”.  Most user roles will save as “Personal Templates”. 

Save Template An object with a pre-set format. The pre-set format can be used as a starting point for a particular application (e.g., a template for entering chemo orders, a template for requesting to add drugs to EPL). The pre-set format can be used to enforce a specific process for completing an action (for quality purposes and/or for policy purposes.): Performs the save function.

Set As Default: When a template is set as default, the search criteria will be pre-populated with the template parameters when the search page is accessed.

Change Search Results Template: Saves modifications to an existing template
