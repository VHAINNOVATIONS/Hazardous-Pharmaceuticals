Manage Search Templates

This page is accessed via the Advanced Search page and allows users to manage or retrieve saved search templates.

Template An object with a pre-set format. The pre-set format can be used as a starting point for a particular application (e.g., a template for entering chemo orders, a template for requesting to add drugs to EPL). The pre-set format can be used to enforce a specific process for completing an action (for quality purposes and/or for policy purposes.) Name Hyperlink: Retrieves the selected search template and populates the search criteria fields

Select Checkbox: Used to perform an action on the selected template

Remove Button:
To remove/delete a saved template from the list:

1.       Select the checkbox for one or more template

2.       Click the “Remove” button.

Set As Default:
Select one (1) search template and click the “Set As Default” button.  When a template is set as default, the search criteria will be pre-populated with the template parameters when the search page is accessed.

Remove as Default:
 Select the default template and click the “Remove as Default” button to clear the default setting.  Default templates are identified by “(Default)” text displayed after the template name.
