Manage Print Templates

Print templates allow the user to customize the field columns displayed in the search results table. 

Customize a Print Template

1.       Click the “Change Search Results Template An object with a pre-set format. The pre-set format can be used as a starting point for a particular application (e.g., a template for entering chemo orders, a template for requesting to add drugs to EPL). The pre-set format can be used to enforce a specific process for completing an action (for quality purposes and/or for policy purposes.)” button on the Advanced Search screen (the Manage Print Template screen is displayed)

2.       Use the arrow and up/down buttons to add and remove fields to display.

3.       Click the “Submit” button to save changes.

Available Fields list: Displays all of the available fields for the entity type.  The list will not display fields that exist in the “Selected Fields” list.

Selected Fields list: Displays the list of field columns which are currently displayed in the template.

Arrow buttons: Add/Remove the highlighted field between the two lists.

Up/Down buttons: Changes the position of the highlighted field up and down.
