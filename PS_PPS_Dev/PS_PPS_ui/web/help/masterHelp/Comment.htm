Comment

This page is used to save the current state of the item as a work in progress.  The user will enter a unique name in the Comment box and then item will be saved.   No changes will be made to the item in the database and no request is generated. 

The Cancel An undo action in a GUI (“cancel” is not an action on an order). button will stop the save process and return the user to the previous screen.

The Submit button will save the item as a work in progress and return the user to the search page.
