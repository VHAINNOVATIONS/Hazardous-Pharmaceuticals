Add NDC

Whether you used a blank product template or an existing product as a template, you will be presented with the “Add New NDC National Drug Code” wizard.  It contains a number of items organized into two field sets (logical groupings).  The field sets or sections include NDC Information and Pill Information.

Each of these field sets or sections contains items that you may or may not complete.  The only items that you must complete are indicated by a red asterisk (*) after the field name.

Select Product button: If you are using a blank template, then you will note a Select Product button.  Click this button to select a product item as the parent association for this NDC.
See the “Select Parent” topic for additional details.

Change Product button: If you are using an existing template, then you will note a Change Product button. This button is available when a parent item already exists.  The button will allow you to change the currently selected parent item.

Cancel An undo action in a GUI (“cancel” is not an action on an order). button: Clicking the Cancel button will terminate the process of adding this new item, and return you to the previous screen.

Submit button: Submits the item addition and presents a confirmation page to the user.
