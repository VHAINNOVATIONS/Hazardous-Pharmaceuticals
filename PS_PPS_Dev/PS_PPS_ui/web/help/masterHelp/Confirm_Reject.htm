Confirm Reject

This page is reachable from when a new item request is being rejected.  The user doing the rejecting must fill in the reasons for the rejection.  The following fields are available.

·         Request A user action that a PEPS Item (OI, Product Item, or NDC Item) addition or modification be considered by the PNM for updates within the PEPS enterprise database. Rejection Reason:  This is a drop down box and the user can select up to three different pre-written reasons.

·         Rejection Reason Comment:  This field is for the user to enter additional rejection reasons.  This is a free text field so the user can enter comments specific to this item.

·         Optional Comments:  This is for optional comments to be recorded.

Cancel An undo action in a GUI (“cancel” is not an action on an order). button: This button will take you back to the edit request screen. 

Submit button: This button will save the rejection to the database.
