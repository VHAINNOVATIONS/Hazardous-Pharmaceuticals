Add Items

New Items may be created via two (2) methods:

Create a new item from a blank template

If you have performed a simple or advanced search, the search results table is displayed.  Below that table, you will note a link to create a new item using a blank template (meaning you are creating an item from scratch rather than using an existing item as a starting point).  To create a new item click the “Create New [item type] (from blank)” link located below the results table as shown below.

Create a new item from an existing template

If you are editing or modifying an existing item, you will note a link to create a new item using the current selected item as a template.  To sue this existing item to create a new one, click the “Create New [item type] (from existing)” link located below the item edit screen.
