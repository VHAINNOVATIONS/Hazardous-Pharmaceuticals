Add Product

Whether you used a blank product template or an existing product as a template, you will be presented with the “Add New National Product” wizard.  It contains a number of items organized into field sets (logical groupings).  The field sets or sections include the following:

·         Define the VA Department of Veterans Affairs Product

·         Define the VA Drug Classes

·         Define the Schedule The frequency by which the doses are to be administered (i.e., Q8H, BID, NOW). Often referred to as Administration Schedule. and Handling

·         Miscellaneous

Each of these field sets or sections contains items that you may or may not complete.  The only items that you must complete are indicated by a red asterisk (*) after the field name.

Select OI Office of Information button: If you are using a blank template, then you will note a Select OI button.  Click this button to select an orderable item as the parent association for this product.
See the “Select Parent” topic for additional details.

Change OI button: If you are using an existing template, then you will note a Change OI button. This button is available when a parent item already exists.  The button will allow you to change the currently selected parent item.

Add New Row button: Some items are organized into collections, and they allow you to include more than one selection or answer.  For example, the Active Ingredient collection includes three associated fields: Ingredient Name, Strength, and Drug Unit.  You will also note that this grouping includes an Add New Row button. Clicking this button will present a blank row and allow you to add another item.

Remove button: Since certain items allow you to provide more than one answer, there must also be a way to remove one or more of the given answers.  Each row in the multiples table concludes with a Remove button.  Selecting this button will remove that entry row.
