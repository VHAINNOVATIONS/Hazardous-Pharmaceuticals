The home page displays the following content:

·         A message from the PBM Pharmacy Benefits Management office

·         The application’s messaging status

·         The user’s “Saved Work in Progress”

·         Helpful links

·         Application A software product that provides a specified functionality. version information

Messaging Status

To perform maintenance tasks, a system administrator may turn off communication with the NDFMS sever.  When this occurs, users will be notified on the home page.  New items and modifications to existing items will be saved to a queue to be process when the connection is re-established.

Saved Work in Progress

The system may automatically save work in progress upon system time-out or, a user may elect to save prior to submitting changes.  Saved sessions will be displayed in the “Saved Work in Progress” table on the home screen.

Comment: A system generated message which summarizes the saved item.  A comment of “AUTO-LOGOUT AUTO-SAVE” designates the work as having been saved upon a user’s session time-out.
Date Modified: A date-time stamp indicating when the item was saved.
