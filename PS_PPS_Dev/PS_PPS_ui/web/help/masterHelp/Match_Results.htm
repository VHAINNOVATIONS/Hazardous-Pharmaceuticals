This page is reachable from either the FDB First DataBank Add or the FDB Search page.  This page is used to take FDB Packaged Drugs and associate them with VA Department of Veterans Affairs Products.  

This page consists of two tables; the FDB NDCs table and the VA Products table.  When the screen is first displayed the FDB NDCs table is populated with those FDB packaged Drugs that were chosen from the FDB Add or FDB Search Page.  All of the FDB Packaged Drugs contain the same GCNSeqNo.   The VA products screen is initially populated with the VA Products that have that same GcnSeqNo.

Blank Template An object with a pre-set format. The pre-set format can be used as a starting point for a particular application (e.g., a template for entering chemo orders, a template for requesting to add drugs to EPL). The pre-set format can be used to enforce a specific process for completing an action (for quality purposes and/or for policy purposes.) button: You must have FDB NDCs selected to use this button.  The button will take the user to the Create new Product screen using a blank template. 

Add to Product button: You must have FDB NDCs selected and a product selected to use this button.  This button enables the user to add the selected FDB NDCs to the selected product.

Use Existing button: You must have FDB NDCs selected and a product selected to use this button.  The button will take the user to the Create new Product screen using the selected product as a template. 

Product Search button:  This button will take the user to a product search screen where the user can search for and select a different product.  That selected product will be added to the VA Products List.

Cancel An undo action in a GUI (“cancel” is not an action on an order). button: This button will take the user back to the original screen.

FDB NDC Fields

·         Select Checkbox:  If the checkbox is disabled, then the NDC National Drug Code in that row is already in PPS-N so cannot be added a second time.   For those that are not grayed out, then a checkbox can be checked or unchecked.  Those checked are the ones selected.

·         NDC: The 11 digit NDC Number.  This field is hyperlinked to show additional FDB DIF Drug Information Framework data for the FDB Packaged Drug.

·         Label Name: The FDB Label Name.

·         Package A set of routines, files, options, templates, security keys, screens, bulletins, functions, help frames, forms, blocks, objects, protocols, dialogues, list templates, windows, and namespaces according to DBA requirements that function as a unit. Size: The FDB Package Size.

·         Package Type: The FDB Package Type.

·         GcnSeqNo: The GCN Generic Code Number Sequence Number.

·         FDB Generic Name A non-proprietary name assigned to an item (e.g., drug, supply).:  The FDB Generic Name.

VA Product Fields

·         Product: The VA Product Name A unique name assigned to each drug product in the National Drug files..  This field is hyperlinked to show the attributes associated with the VA Product.

·         Item A generic type representing one conceptual entity consisting of a collection of data fields. For example, a medication may contain the following collection of data fields: Product Name, Generic Name, Trade Name, Dosage Form, NDC, etc. Conceptual entities include Medications, Supplies, and Investigationals. Status: Shows the Status of the VA Product:  Could be APPROVED, PENDING or REJECTED.

·         GcnSeqNo: The GCN Sequence Number associated with the VA Product.

·         Use:  A radio button that is used to select a specific VA Product.
